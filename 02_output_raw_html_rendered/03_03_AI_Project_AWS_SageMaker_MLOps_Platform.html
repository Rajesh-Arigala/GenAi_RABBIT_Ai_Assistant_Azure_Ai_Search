End-to-End AWS SageMaker MLOps

This project demonstrates a complete enterprise‑grade MLOps platform on AWS using Amazon SageMaker, AWS CDK (Infrastructure as Code), GitHub Actions, and OIDC authentication. It automates the ML lifecycle across infrastructure, training with quality gates, and registry‑driven production deployment.

Secure CI/CD

OIDC‑based GitHub Actions avoids static cloud credentials. Deployments are scoped with least‑privilege IAM and can be cleanly separated per environment.

Governed promotion

Models are versioned in SageMaker Model Registry and promoted only after manual approval gates to reduce risk.

Automation first

Infrastructure, training, evaluation, registration, and deployment are automated so teams can iterate quickly with traceability.

Business Problem

Notebook models are easy. Production ML is hard without a repeatable delivery platform.

The pain

Many teams train models manually and deploy inconsistently. They lack governance, monitoring, and secure automation—leading to delays, quality issues, security risks, and high operational overhead.

Impact: slow releases, fragile deployments, limited auditability, and expensive rework.

The solution

This project delivers a reusable AWS MLOps foundation that standardizes training, evaluation, registry governance, and production deployment using native AWS services.

Outcome: consistent, reproducible, and governed ML operations across environments.

Cost-Aware AWS MLOps (FinOps)

Designed with FinOps principles: scale-to-zero jobs, on-demand endpoints, and quality gates to prevent wasteful deployments.

Key cost controls

  • Scale-to-zero training and processing jobs
  • On-demand SageMaker endpoints (easy to create/delete)
  • Infrastructure as Code for quick spin-up/teardown
  • Quality gates to prevent wasteful deployments
  • Batch Transform for non-urgent workloads
Main cost driver: real-time SageMaker endpoints. Delete endpoints after testing for portfolio/demo projects to stay under ~$60/month.

Estimated monthly cost profile

Scenario Description Approx. monthly cost
Portfolio / Demo No running endpoint $15 – $60
Active Development Occasional training + testing $50 – $150
Production Pilot One real-time endpoint + retraining $120 – $280
Small Production Endpoint + monitoring + weekly retraining $200 – $500
Recommendation: use Batch Transform where latency is not critical; keep real-time endpoints only when the business requires low-latency predictions.
